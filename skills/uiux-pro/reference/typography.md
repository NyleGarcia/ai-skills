# Typography Reference

## Core Rules
- **No Defaults:** Avoid Inter, Roboto, Arial, Times New Roman. 
- **Pairings:** Contrast serif with sans-serif, or geometric with humanist.
- **Scale:** Use modular scales (e.g., Major Third - 1.250). Base 16px.
  - h1: 3.052rem
  - h2: 2.441rem
  - h3: 1.953rem
  - p: 1rem
  - small: 0.8rem
- **Line Height (Leading):**
  - Headings: 1.1 - 1.2
  - Body: 1.5 - 1.6
- **Line Length (Measure):** 45-75 characters per line. Max-width on paragraphs.
- **Weights:** Skip weights to build contrast (e.g., Light 300 with Bold 700). Never use semi-bold next to bold.
- **Features:** Enable OpenType features (ligatures, tabular nums for data).

## Anti-Patterns
- Too many fonts (>2 families).
- Centered text blocks longer than 3 lines.
- Justified text.
- Gray text on colored backgrounds.
