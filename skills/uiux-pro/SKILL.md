# UI/UX Pro Design Intelligence

Advanced reasoning engine. Fights generic LLM design bias.

## Core Anti-Patterns (NEVER DO THIS)
- NO overused fonts (Arial, Inter, system defaults). Use curated typography.
- NO gray text on colored backgrounds. Use tinted neutrals.
- NO pure black (#000000) or pure gray. Always tint with primary hue.
- NO excessive cards or card-in-card nesting. Use whitespace/dividers.
- NO bounce/elastic easing. Use purposeful, modern motion.
- NO generic purple/blue gradients unless specifically requested.

## Interaction Modes & Commands
- `/craft` or `APPLIER`: Phased build (Foundation -> Structure -> Typography -> Composition -> Color -> Technical).
- `/audit` or `CHECKER`: Tech checks (a11y, performance, responsive, SEO). Stop at first failure.
- `/critique`: UX review (hierarchy, clarity, emotion).
- `/normalize`: Apply design tokens, fix spacing inconsistencies.
- `/polish`: Final alignment pass before shipping.
- `/distill`: Strip design to essence.
- `/harden`: Error states, edge cases, i18n, onboarding.
- `/colorize`: Introduce strategic color.
- `/animate`: Add purposeful motion.

## Design Systems Generation
When starting new UI:
1. Define Pattern (Hero-Centric, Dashboard, etc.).
2. Define Style (Soft UI, Brutalism, Neumorphism, Glassmorphism).
   - **Soft UI Evolution:** Soft shadows, subtle depth, organic shapes. Best for wellness/premium brands.
3. Define Color Palette (Primary, Secondary, CTA, Background, Text - use OKLCH/HSL).
4. Define Type Scale (Modular scaling, Header pairings).

## Learned Preferences (History)
- **Icons:** NEVER use emojis as icons. Prefer SVG (Heroicons/Lucide).
- **Interactions:** Always add `cursor-pointer` to clickables. Use smooth transitions (150-300ms).
- **Motion:** `prefers-reduced-motion` MUST be respected.
- **Anti-Patterns:** Avoid "AI purple/pink gradients", harsh animations, and neon contrast in light mode.

## Reference Areas
- **Typography:** Font pairing, modular scales.
- **Color & Contrast:** OKLCH, tinted neutrals, dark mode, WCAG.
- **Spatial Design:** 4px/8px grids, visual rhythm, hierarchy.
- **Motion:** Easing curves, staggered intro.
- **Interaction:** Focus states, loading patterns, form validation.
- **Responsive:** Mobile-first, fluid typography, container queries.
