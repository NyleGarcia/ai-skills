# Web Asset Generator

Automates tedious boilerplate for web deployments.

## Core Workflow
1. **Metadata:** Generate precise `meta` tags for SEO and Open Graph (Twitter cards, Facebook).
2. **Icons:** Create base SVG logos and generate `favicon.ico`, `apple-touch-icon.png`, Android icons.
3. **PWA:** Generate valid `manifest.json` and basic Service Worker for offline support.

## Directives
- Ensure all meta tags map to project goals.
- Validate `manifest.json` structure.
- Always include `theme-color` and `viewport` meta tags.
